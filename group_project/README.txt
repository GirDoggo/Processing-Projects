The main menu is navigated with the up and down arrow keys. Pressing the enter key on any of the main menu elements, except the Name,
will trigger their functions. To get a new username, simply type it into the Name box. Backspace will delete everything in the name
space. Pressing play will start the game. Pressing settings will open the top ten high scores and the associated usernames. Pressing
exit will exit the program.

In the actual game, only one button, spacebar, controls the butterfly -- by making it fly upwards. After dying, the game over screen
let's you return to the main menu by pressing the spacebar.
